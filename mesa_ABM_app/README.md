
# Mesa ABM + Solara UI (Fixed)

Start from project root:

```bash
pip install -U -r requirements.txt
solara run solara_app.py
```

Use the **Model parameters** card to adjust space/time/merge and phenotype parameters, then
click **Apply parameters & Reset** to re-initialise the model with your settings.

Use **Run/Pause/Step** in **Simulation controls**. The **space view** should show coloured
clusters sized by radius; **histogram** updates each step.
